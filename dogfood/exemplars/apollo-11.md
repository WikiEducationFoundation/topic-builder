# Menu card

**Shape axes**

- structural: named event with concentric program / agency / people reach
- scale: hundreds (~150–300)
- layered_shape: concentric — flagship event + parent program +
  agencies / facilities + crew + ground-side personnel + cultural
  reception
- non-Anglosphere depth: moderate — Cold War context, Soviet response,
  international scientific community
- biography density: medium-high (astronauts, engineers, flight
  controllers, contractors, political figures)
- canonical category coverage: partial — `Category:Apollo 11` is core
  but reach lives across Apollo program / Kennedy Space Center / 1960s
  spaceflight categories
- recall_ceiling_driver: stitching across concentric layers — the
  flagship category alone caps recall around 50%

**Doesn't apply when:** there's no clear parent program / framing
institution to stitch outward to; OR the flagship category already
covers ~all the reach (no concentric layers).

**Shape (prose).** A well-known historical event with a small flagship
core and significant peripheral reach via the parent program, the
agencies and facilities that executed it, and the people involved.
Recall depends on stitching across the concentric layers, not just
working the event category.

**Summary.** Best run reached ~250 articles at ~95% precision and ~80%
recall by combining the event-named category with a navbox harvest of
the parent program template, then sweeping agency / facility
categories (Kennedy Space Center, North American Aviation contractors,
Mission Control personnel) for the broader operational context. The
move that consistently separates strong runs from weak ones is
recognizing this is a *concentric* topic — the flagship category
alone caps recall around 50%.

**High-leverage moves**:

- **Navbox harvest on the parent program template** is the single
  biggest reach move. The flagship event's own category is too narrow;
  the program template stitches together the agencies, missions,
  spacecraft, and people that the event sits inside.
- **Agency / facility category sweeps** for ground-side personnel and
  contractors. People like flight controllers and industrial-base
  contractors sit outside the event category but are clearly on-topic
  for a complete picture.
- **`P39` (position held)** Wikidata probe for institutional roles
  (NASA astronaut, NASA flight director) surfaces biographies missed
  by category routes.
